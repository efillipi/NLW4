## 👍🏻 Como baixar

```bash

    // Clonar repositório
    $ git clone https://github.com/efillipi/NLW4.git

    // Acessar diretório
    $ cd permissions_users

    // Instalar dependências
    $ yarn

    // Iniciar projeto
    $ yarn start
```
